## V3-04 Core Pages UI Polish

- **Admin (students / teachers / attendance / finance):** core lists now use dashboard shell cards, filter toolbars, clearer status badges, and empty-state rows to reduce scanning friction during demos.
- **Teacher (attendance / grades):** attendance capture and grade entry are now grouped into card sections with class/date filters and explicit empty guidance when no class or assessment is selected.
- **Parent / Student (grades / finance):** grade and invoice tables now highlight key values with badges and formatted currency, while keeping existing role-scoped data rules unchanged.
- **Messaging (inbox / thread / announcements):** inbox and announcement pages now follow the same card layout as dashboards for visual coherence across product areas.
