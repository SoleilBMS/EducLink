# Demo Flow

## Purpose
This document outlines the end-to-end demo narrative for EducLink.

## Public entry point
- Commercial landing page: `/`
- Demo guide page: `/demo`
- Authentication page: `/login`

The landing page is intentionally **pilot/demo-ready** messaging for decision-makers (not presented as production-ready).

## Demo narrative (quick path)
1. Start on `/` to present positioning and key value proposition.
2. Use CTA **Voir la démo** to move to `/demo`.
3. Use demo accounts listed on `/demo` and access `/login` to simulate role journeys.
